 Project title : Customer Churn Prediction Web App

 Description of the model : This is a machine learning web application that predicts whether a customer will churn or stay based on their demographic and service-related features.I have trained this model using Logistic regresson ,Random Forest and XGBoost . 
 Links:
      Live App: https://churn-prediction-app-vxocs4abjh8jsairhtqnyp.streamlit.app/
      GitHub Repo: https://github.com/Rabeya29472/churn-prediction-app
Features of the model:
     Predicts customer churn in real-time
     User-friendly Streamlit interface
     Handles categorical and numerical inputs
     Trained using multiple ML models (Random Forest selected as final model)

Tech Stack: Python,Pandas,Scikit-learn,Streamlit,oblib

Datset Informations : These are the featurs of the dataset-Age, Gender,Tenure,Monthly Charges, Contract Type,Internet Service,Tech Support

Model Informations: 
     Random Forest Classifier has been used as the final model
     Other models like Logistic Regression and XGBoost has been tested and compared 
     Model selected based on performance comparison
